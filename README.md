# Calculator App

A calculator that supports positive, negative, decimal numbers, and parentheses.


```sh
$ git clone https://github.com/itzsarahhx3/calculator-app.git
```

```sh
$ cd calculator-app
```

```sh
$ npm install
```

## Run Locally (http://localhost:3000)

```sh
$ npm start
```

## Run Tests

```sh
$ npm test
```
